# Vue Scaffolding

## Global Install
### vue-cli
```
npm install -g vue-cli
```
### Firebase tools
```
npm install -g firebase-tools
```
### Init vue project
```
vue init webpack collex
```

##  Local Install
### npm install
```
cd collex
npm install
npm run dev
```
### Install firebase/vuefire
```
npm install firebase vuefire --save
```
